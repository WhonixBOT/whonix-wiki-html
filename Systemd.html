Systemd User Environment Variables Configuration[edit]

x. 

mkdir -p ~/.config/environment.d/

x. Open file ~/.config/environment.d/electrumx.conf in a text editor of your choice as a regular, non-root user.

If you are using a graphical environment, run. 

mousepad ~/.config/environment.d/electrumx.conf

If you are using a terminal, run. 

nano ~/.config/environment.d/electrumx.conf

x. Paste the following contents. 

COIN=Bitcoin
DB_DIRECTORY=~/.electrumx
DAEMON_URL=username:password@127.0.0.1
SERVICES=rpC://localhost
PEER_DISCOVERY=self

x. Save.

x. 

systemctl --user daemon-reload
