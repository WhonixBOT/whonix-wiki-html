Recommended VirtualBox Version for use with Whonix ™

From Whonix

< VirtualBox



It is highly recommended to use the recommended VirtualBox version for use with Whonix ™. Otherwise there can be issues. [1]

Recommended VirtualBox version.

  • Whonix ™ VirtualBox Stable Version: 6.1.20
  • Whonix ™ VirtualBox Testers Only Version: 6.1.20

Instructions for installing VirtualBox on Debian Linux can be found for example on the Whonix ™ for VirtualBox with XFCE page, which contains the following text.

* Linux : please press expand on the right side.

Once expanded, the recommended instructions will explain how to install VirtualBox.

Footnotes[edit]

  1. ↑ Whonix ™ 15.0.1.7.3 was built and tested with VirtualBox version 6.1.20. Newer VirtualBox not listed on the stable version of Whonix ™ for VirtuallBox, testers-only version of Whonix ™ for VirtuallBox or otherwise in Whonix ™ News are untested by Whonix ™ developers and can cause issues such as instability (VirtualBox VMs not starting [archive], VMs suddenly closing, host freezing, VM black screen). Users are advised to use the wait until newer versions of VirtualBox are recommended for use with Whonix ™.

    Newer VirtualBox versions are hard to install on Debian buster. Perhaps the VirtualBox host software is available (even as a Debian package) from virtualbox.org repository VirtualBox guest additions are not available as a Debian package. Using an older version of VirtualBox guest additions (such as the pre-installed ones) might lead to broken functionality such as an inability to resize the VM or instability. Installation of VirtualBox guest additions from ISO might also cause issues. [archive]
