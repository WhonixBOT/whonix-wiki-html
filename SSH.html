SSH


Documentation for this is incomplete. Contributions are happily considered!

Use stronger ssh keys. 

ssh-keygen -t ed25519

Prefer SSH servers available through a Tor Onion Service for stronger encryption and authentication.

mosh might be useful, however since it requires UDP, using it over Tor is non-trivial, see Tunnel UDP over Tor.

Beware of keyboard keystroke and computer mouse based deanonmization as explained under Surfing Posting Blogging.
