Qubes/Clone VMs

< Qubes

Cloning Qubes VMs is a simple process when using Qubes VM Manger.

Qubes VM Manager -> VM to be Cloned -> Clone qube -> Enter name for Qube clone





[1]

Footnotes[edit]

  1. ↑ When prompted, users should give the newly cloned VM a name that is not easily confused with other VMs. Ensuring there is no ambiguity when choosing names is especially important when cloning TemplateVMs. This will minimize the chances of a user basing an AppVM on the wrong template. There could be serious security concerns if a user based a trusted AppVM on the wrong template such as a TemplateVM with untrusted packages.
