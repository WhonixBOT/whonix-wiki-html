Skipping Postinst Scripts[edit]

Advanced users only!

Let's say for example you don't like the /usr/share/whonix/postinst.d/70_desktopicons postinst script creating Desktop icons each time an update from Whonix ™ apt repository comes.

Once you identified which postinst script you don't like, just run 

export WHONIX_BUILD_SKIP_SCRIPTS+=" 20_sanity_checks "

before running "apt-get dist-upgrade".

Or you could add them to the WHONIX_BUILD_SKIP_SCRIPTS environment variable separated by spaces to /etc/environment, example. 

WHONIX_BUILD_SKIP_SCRIPTS="70_desktopicons 70_knetattach 70_whonix_apt_key"

After changing /etc/environment, you need to reboot.

In case this feature is of interest to you, a solution for not requiring reboot and adding them to a config file in /etc/whonix.d could be easily and quickly added. Just make a feature request in that case.
