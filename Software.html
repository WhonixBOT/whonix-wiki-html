Warning[edit]

Accessibility[edit]

On-screen Keyboard[edit]

Florence [archive] is an extensible, scalable, virtual (on-screen) keyboard. [1]

To install it: Start menu → System → Xfce Terminal

1. Update the package lists. 

sudo apt-get update

2. Upgrade the system. 

sudo apt-get dist-upgrade

3. Install the florence at-spi2-core package. 

sudo apt-get install florence at-spi2-core

The procedure is complete.

Terminal Editor[edit]

Open File with Root Rights[edit]

Prefer sudoedit for better security. [6] Replace /path/to/file with the actual file name. 

sudoedit /path/to/file

To use a different editor. Replace mousepad with the name of the editor you wish to use. 

SUDO_EDITOR=mousepad sudoedit /path/to/file

An alternative would be to install these tools lxsudo and mousepad so you can keep copying and pasting these instructions.

1. Update the package lists. 

sudo apt-get update

2. Upgrade the system. 

sudo apt-get dist-upgrade

3. Install the --no-install-recommends sudo lxsudo mousepad package. 

sudo apt-get install --no-install-recommends sudo lxsudo mousepad

The procedure is complete. 

lxsudo mousepad /path/to/file

Publishing[edit]

Scribus [archive] (Debian package [archive]) is an open source desktop page layout accessible from the desktop menu (Applications → Graphics). It can be used for many tasks; from booklets design to newspapers, magazines, newsletters and posters to technical documentation.

Scribus has sophisticated page layout features like precision placing and rotating of text and/or images on a page, manual kerning of type, bezier curves polygons, precision placement of objects, layering with RGB and CMYK custom colors. The Scribus document file format is XML-based. Unlike proprietary binary file formats, even damaged documents can be recovered with a simple text editor.

To install it: Start menu → System → Xfce Terminal

1. Update the package lists. 

sudo apt-get update

2. Upgrade the system. 

sudo apt-get dist-upgrade

3. Install the scribus package. 

sudo apt-get install scribus

The procedure is complete.

scurl: SSL Command Line Downloader[edit]

To securely download files or webpages from the Internet on the command line, scurl is pre-installed in Whonix ™. scurl should be preferred over wget, since the former enforces strong encryption and is less buggy.

To invoke scurl to download a file, use the following syntax and replace the example URL with the file's location. 

scurl --remote-name https://dist.torproject.org/torbrowser/8.5.1/tor-browser-linux64-8.5.1_en-US.tar.xz

This will download tor-browser-linux64-8.5.1_en-US.tar.xz to the current working directory.

Refer to the scurl entry for further examples and a complete description of this tool.
